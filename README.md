# ft_ls
ft_ls 42 school project

## Description
Recoding the shell ls command

## Features
- Supported flags: -l, -R, -a, -r, -t and -G
- Management of ACL and extended attributes
- Management of the columns without the option -l (output sorted by columns)

## HOW TO USE

```
$> make
$> ./ft_ls <args>
```
